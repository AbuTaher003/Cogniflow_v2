# 🚀 About CogniFlow

**CogniFlow** is a modern AI-powered Student Operating System designed to help students manage their entire academic journey inside a single platform.

Instead of using multiple disconnected tools, students can use CogniFlow to:

* Plan studies
* Manage tasks
* Track academic progress
* Prepare for competitive programming
* Build resumes
* Track internships
* Analyze productivity
* Receive AI-powered assistance
* Manage subscriptions
* Access personalized dashboards

The vision of CogniFlow is to become the **"Operating System for Students."**

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/AbuTaher003/Cogniflow_v2.git
```

Enter the project:

```bash
cd Cogniflow_v2
```

Install dependencies:

```bash
npm install
```

Create environment variables:

```env
NEXT_PUBLIC_SITE_URL=
NEXT_PUBLIC_APP_NAME=
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPER_ADMIN_EMAIL=
```

Run development server:

```bash
npm run dev
```

Create production build:

```bash
npm run build
```
